Amethyst Easyche?Amethyst Easywhat?
Easyweb e'  un linguaggio che consente lo sviluppo di sistemi interattivi su internet in modo molto semplice. Easyweb is a language for the develop of interactive systems on internet in a very simple way
E' possibile infatti, con pochissime riche di codice, creare delle pagine che possono comunicare con l' utente, permettere l' upload di files, mantenere memoria dei propri dati in tabelle di database che possono essere interrogate in maniera estremamente semplice e naturale. With few lines of code you can create pages that can dialog with users, upload files, keep memori of the user's data in database tables that you can query in a easy and natural way.
A tutto questo e' poi possibile aggiungere l' interfaccia grafica, con la facciata che voi volete, e magari modificabile quando volete, con una sola istruzione. To that you can add a graphical interface, with the face that you want, and if you want easy to change, with a single instruction.
Questo e' generalmente tutto quello che puo' essere utile per lo sviluppo di un sistema informativo..  a parte magari l' analista e il programmatore, che non sono forniti nel pacchetto, perche' non ci entravano (non sono riuscito a zipparli). This is generally all what you need do develop an informative system. Except for the programmer and the analyst that aren't in the package because they didn't zip.
